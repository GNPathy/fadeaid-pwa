# FadeAid PWA — User Manual

## 1. First-Time Setup

### Add to Home Screen (Recommended)

**iPhone (Safari):**
1. Open the FadeAid URL in Safari.
2. Tap the **Share** button (box with arrow pointing up).
3. Tap **Add to Home Screen** → **Add**.

**Android (Chrome):**
1. Open the FadeAid URL in Chrome.
2. Tap ⋮ → **Add to Home Screen**, or accept the install banner.

> **Note:** Haptic vibration works on Android. On iPhone, FadeAid uses a color-flash on the tapped band instead — this works well for eyes-up operation.

---

## 2. Setting Up Context Lists

Open the **Settings** tab (⚙️ icon).

1. Tap **Configure Source List**.
2. **Students**: comma-separated names (e.g., `Alex, Sam, Jordan`).
3. **Subjects**: comma-separated (e.g., `Math, Life Skills, Reading`).
4. **Sub-Topic Mappings**: per-subject suggestions (e.g., Math → `Addition, Subtraction`).
5. **Aide Name** and **Classroom**: auto-added to every session report.
6. Tap **Save**.

---

## 3. Creating Student Configurations

1. In Settings, tap **Set-up Sessions**.
2. Pick a **Student** and **Subject**.
3. Set **Target Goals** for Verbal, Visual, and Gestural prompts.
4. Tap **Save**.

> *Goals apply at the Student × Subject level — all sub-topics under "Math" for "Alex" share the same goals.*

---

## 4. Starting a Class

1. Open the **Sessions** tab.
2. Tap **⚙ Start Class**.
3. Select **Student × Subject**.
4. Select **Sub-Topic** (or type a custom one).
5. Optionally enter **Session Goals** or **IEP Notes**.
6. Tap **▶ Start Class**.

---

## 5. The 4-Band Tracking Screen

| Touch Region | Action |
|---|---|
| **Right half (+)** | Increment — log a prompt or response |
| **Left half (−)** | Decrement — undo the last logged event |
| **Center** | Current count (read-only) |

| Band | Color | Tracks |
|---|---|---|
| Response (top) | Green / Red | Correct ✅ and Incorrect ❌ responses |
| Verbal Prompts | Blue | Spoken instructions or cues |
| Visual Prompts | Purple | Pictures, objects, modeling cues |
| Gestural Prompts | Teal | Pointing, nodding, gestures |

**Confirmation:** Android vibrates; iPhone flashes the band color + audio pop.

---

## 6. Ending a Class

1. Tap **■ End Class** (top-right during tracking).
2. Optionally add **Session Notes**.
3. Tap **Save & End**.

---

## 7. History Tab

- All past sessions, newest first.
- Tap a card to expand full details.
- Tap **📄 Export PDF** for a professional single-session report.

---

## 8. Analytics Tab

- Filter by Student and/or Subject independently.
- **Weekly Summary**: prompt counts by day (Mon–Fri).
- **Cumulative Performance**: aggregated totals across all sessions.
- **📊 Export CSV**: downloads a `.csv` of all session data.

---

## 9. Privacy & Security

- **100% Offline**: Nothing is ever sent to a server.
- **No Sensors**: No camera or microphone used.
- **FERPA Ready**: Go to **Settings → Clear All Data** to wipe everything permanently.

---

## 10. Troubleshooting

| Issue | Solution |
|---|---|
| Screen turns off during class | Tap **Start Class** before leaving the setup screen; check screen timeout in phone Settings |
| Audio not working | Check device volume; ensure tab is not muted; check iPhone Silent Mode switch |
| Data missing in History | Ensure you tapped **Save & End** — open sessions don't appear until closed |
| "Add to Home Screen" not appearing | iPhone: use Safari only. Android: use Chrome only |
